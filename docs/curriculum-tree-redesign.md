# 교과목 트리 페이지 최종 스펙 (v5 — 통합 그리드 · 파랑 채움 + 주황 택1 확정)

> 대상: 교육·학사 > 교과과정 > 기계공학부 교과목 트리
> 확정 방향: 하나의 8열 통합 그리드(레인 구조).
> 필수 = 진한 파랑 채움(확실한 가시성), 택1 = 주황(화면 유일의 웜톤 포인트).
> "눈에 확 띄는 것"은 기능 요건 — 택1은 놓치면 안 되는 정보이므로 보색 대비를 쓴다.

## 0. 색 시스템 (최종)

| 역할 | 값 |
|---|---|
| 구조: 레인 바, 제목, 활성 탭 | `#0c2948` |
| 필수과목 카드 | 배경 `#1769a8` + 흰 글자 (테두리 없음) |
| 선택과목 카드 | 흰색 + 테두리 `1px #d8dee5` + 글자 `#182331` |
| 택1 그룹 박스 | 테두리 `1.5px #e8630a` + 배경 `#fff8f1` + "택 1" 배지 `#e8630a` |
| 링크·호버·선수과목 하이라이트 | `#1769a8` |
| 학번 안내 배너 | 액센트 `#b42318`, 배경 `#fdf3f2` |

- **주황 사용 범위 제한 (중요):** 주황은 택1 그룹 요소(테두리·배지·그룹 제목)에만.
  면 채움 금지 — 테두리와 배지만. 다른 어떤 요소에도 웜톤 사용 금지.
- 필수 카드 내부의 선수 메타 텍스트는 흰색 75% (`rgba(255,255,255,.75)`).
- 그림자 전면 금지. 구분은 채움/테두리로만.

## 1. 마감 규칙 ("도표처럼 딱 떨어지게")

- 모서리 radius **3px 통일** (카드·택1 박스·레인 바 전부 동일)
- 카드 min-height **64px 고정** (2줄 과목명 기준 — 높이 들쑥날쑥 금지)
- 간격 **8px 하나로 통일** (카드 사이 = 열 사이 gap)
- 과목명 15px/600, 선수 메타 12px `#5b7b96`, `word-break: keep-all`, 최대 2줄

## 2. 페이지 구조 (위→아래)

1. 제목 "기계공학부 교과목 트리" + 설명 1줄, 우측 "원본 체계도 PDF" 버튼
2. 학번 안내 배너 (기존 유지) + "학번별 졸업요건 보기 →"
3. 범례 1줄: 연파랑 카드 = 필수 / 흰 카드 = 선택 이수 / 점선 박스 = 택 1 / 과목 클릭 = 선수과목 표시
4. 학기 탭 (sticky): 전체 · 1-1 ~ 4-2, 활성 = 네이비 필. 선택 시 해당 열 외 opacity 0.35
5. 통합 그리드 (아래 3절)
6. 스페셜 트랙 배너 (배경 `#eaf4fb`, 기존 유지)
7. 교양 섹션 (아래 4절)

## 3. 통합 그리드 (핵심)

**페이지 전체에서 8열 축은 하나만 존재한다.** 섹션별 별도 그리드 금지.

```
[진입 배지 행]  ↓2학년 진입 전 필수(2-1 위)   ↓3학년 진입 전 필수(3-1 위)
[학기 헤더 행]  1-1  1-2  2-1  2-2  3-1  3-2  4-1  4-2   ← 페이지에 1회만
[MSC 레인]     수학1  수학2  수학3  수학4  [택1그룹]      (3-2~4-2: "해당 없음")
               물리1  물리2  정보처리
               화학1  화학2
[전공 레인]                고체역학 유체역학 [전공 심화 패널: 3-1~4-2 span 4]
                          열역학   동역학    ├ 1학기 개설 (좌열+우열 전체 과목)
                          컴응기설 실험(1)   └ 2학기 개설 (좌열+우열 전체 과목)
[4학년 행]     창의제품설계  연구논문  |  기계공학세미나 (1)  (2)
```

- grid-template-columns: `56px repeat(8, 1fr)` — 모든 레인이 공유
- 레인 머리: 네이비 바에 "MSC" / "전공" 세로 라벨 1개만. 중복 소제목
  ("MSC 교과목", "MAJOR FOUNDATION", "전공 기초 · 2학년" 등) 전부 삭제
- 전공 심화 패널: 전공 레인의 3-1~4-2 열 (`grid-column: span 4`),
  내부 "1학기 개설 | 2학기 개설" 2단, 과목 전체 표시 (축약 금지)
- 진입 배지: 학기 헤더 행 **위** 별도 행 (열 라벨 가림 금지),
  배경 `#eaf4fb`, 테두리 `1px #b9cce6`, 글자 `#0c2948` 12px
- 세로 점선: 1-2/2-1, 2-2/3-1 경계에서 MSC·전공 레인 관통, `2px dashed #8fb0cc`
- 택1 그룹: 3-1 열 폭 안에 정확히 (돌출 금지), §0의 주황 박스 스타일,
  "택 1" 배지 = 주황 `#e8630a` 배경 + 흰 글자, 옆에 "1과목 선택" 12px `#9a5b23`
- 빈 구간: 빈 카드·라벨 반복 금지. 레인 배경만 잇고 "해당 없음" 12px `#9aa8b6` 1개 (선택)
- 필수 카드의 "필수" 배지 시각 표시 제거 (aria-label에는 유지) — 연파랑이 필수를 뜻함

## 4. 교양 섹션 (v2 유지 + 색만 교체)

- 상단 2카드: "공통 교양 · 필수 이수" 체크리스트 + "공학 관련 권장 교양" 칩
  (캡션: "졸업 필수요건 아님 · 이수 권장")
- 영어 트랙 택1: §0의 주황 박스 스타일 (교과 그리드의 택1과 동일 문법)
  { 대학영어 Ⅰ+Ⅱ | 고급대학영어 Ⅰ + 대학영어 Ⅱ }
- "교양 영역 이수요건" 통합 카드 1개:
  - 영역 칩 7개 1회만 표시, "정보와기술" 칩 = 점선 테두리 + "21학번까지" 태그
  - 요건 스트립: 21학번 이전(배경 `#eef4f8`, 숫자 "5") / 22학번 이후(배경 `#dbeaf5`, 숫자 "4")
    — 숫자 28px/800 `#1769a8`
  - 각주: "* 22학번부터 정보와기술 영역이 제외되어 이수 영역이 5개에서 4개로 변경됩니다."

## 5. 인터랙션

- 과목 카드 클릭 → 선택 카드 `0 0 0 2px #0c2948` 아웃라인,
  선수과목 카드 `0 0 0 2px #e8630a` + "선수과목" 라벨. Esc/재클릭 해제
  (선수 하이라이트도 주황 — "지금 주목할 것"이라는 동일한 의미)
- aria-live: "○○의 선수과목: ○○ (n학년 n학기)"
- 호버: 흰 카드 테두리 `#9db4d6` + 배경 `#f8fafd`, 필수 카드 배경 소폭 어둡게 (`#125687`)
- 전환 0.15s만 허용. 스크롤 트리거 애니메이션 금지

## 6. 반응형

- 1024px: 열 간격 8→6px, 유지 불가 시 그리드 컨테이너만 가로 스크롤 (탭 동기화)
- 768px 미만: 그리드 폐기 → 학년 아코디언 (기존 모바일 구조 유지),
  진입 기준선은 아코디언 사이 가로 배지, 카드 스타일은 §0 동일
- 터치 영역 44px 이상

## 7. 접근성

- 카드 aria-label: "공학수학 4, 2학년 2학기, 필수, 선수과목: 공학수학 3"
- 필수/택1 구분은 색+형태(채움/점선) 이중 — 색맹 대응 충족
- 키보드: Tab 순서 = 학기 탭 → 레인별 카드 학기순, Enter/Space 선택, Esc 해제

## 8. 금지

- 주황을 택1 요소·선수과목 하이라이트 외에 사용하는 것 (골드 등 다른 웜톤 전면 금지)
- 주황 면 채움 (테두리·배지만 허용)
- 연파랑 채움 필수 카드로의 회귀 (필수는 `#1769a8` 진한 채움 확정)
- 그림자, 그라데이션, 아이콘 장식
- 섹션별 8열 그리드 중복, 열 라벨 반복, 빈 카드 렌더링
- 새 문구 창작

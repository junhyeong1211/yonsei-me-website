# 교과목 트리 페이지 리디자인 스펙

> 대상: 교육·학사 > 교과목 체계도 > 기계공학부 교과목 트리
> 방향: 원본 이미지(PDF 체계도)를 이미지로 다시 만들지 않고,
> 학기 열 그리드 기반의 인터랙티브 웹페이지로 구현한다.
> 원본의 과목명·필수여부·택1·선수관계·학번별 요건은 그대로 유지한다.

## 0. 전제

- 과목 데이터는 코드와 분리한다:
  `{ id, 과목명, 영역(msc|major|ge), 학기, 필수여부, 선수과목 id[], 택1그룹 }`
- 페이지 우상단에 "인쇄용 PDF 다운로드" 버튼 제공 (원본 체계도 PDF 링크).
- 색 역할 규칙: 파랑 채움 = 필수 / 흰 카드 = 선택 이수 / 주황 = 택1 전용 /
  네이비 = 구조(제목·라벨). 이 외 용도로 색 사용 금지.

## 1. 페이지 구조 (위→아래)

1. 브레드크럼 + 페이지 제목 "기계공학부 교과목 트리" (32px/700, 네이비)
2. 설명 1줄 + 졸업요건 안내 배너
   - 배너: 좌측 4px 액센트 `#b42318`, 배경 `#fdf3f2`, 본문 14~15px
   - 문구: "학번마다 교과목 체계가 다르니, 상세 정보를 위해 각 학번의 졸업 요건을 확인하시기 바랍니다."
   - 우측에 "학번별 졸업요건 보기 →" 링크
3. 범례 한 줄: 필수(파랑 칩) / 선택 이수(흰 칩) / 택 1(주황 테두리 칩) / 선수과목(→, "과목 클릭" 안내)
4. 학기 탭 (sticky, top 0): 전체 · 1-1 · 1-2 · 2-1 · 2-2 · 3-1 · 3-2 · 4-1 · 4-2
   - 필(pill)형, 활성 = 네이비 배경 + 흰 글자
   - 특정 학기 선택 시 해당 열만 100%, 나머지 열 opacity 0.35 (레이아웃 유지)
5. MSC 섹션 → 전공 섹션 → 교양 섹션 (각각 흰색 카드, 좌측 네이비 세로 라벨 바 56px)

## 2. 학기 그리드 (MSC · 전공 기초 공통)

- 8열 그리드 (1-1 ~ 4-2), 열 간격 16px, 각 열 상단에 학기 라벨 13px
- 학년 진입 기준선: 1-2/2-1 사이, 2-2/3-1 사이에 세로 점선
  `2px dashed #7c93b8` + 상단 알약 배지(배경 `#eaf4fb`, 글자 네이비 12px/700)
  - "2학년 진입 전 필수" / "3학년 진입 전 필수"
  - 빨간 화살표 사용 금지 (경고가 아니라 구조 정보)

## 3. 과목 카드

공통: 최소 높이 56px, 모서리 8px, 패딩 12px 14px, 과목명 15px/600,
`word-break: keep-all`, 최대 2줄 말줄임, 카드 전체 클릭 영역, `tabindex=0`.

| 상태 | 스타일 |
|---|---|
| 필수 | 배경 `#1769a8`, 흰 글자, 우상단 "필수" 배지 11px (흰색 18% 배경) |
| 일반 | 흰 배경, `1px #d8dee5`, 글자 `#182331` |
| 택1 그룹 | 그룹 박스: `1.5px #e8630a` 테두리 + 배경 `#fff8f1` + "택 1" 주황 배지, 내부 카드는 일반 스타일 |
| 선수과목 보유 | 카드 하단에 "선수: ○○" 12px 메타 (필수 카드는 흰색 75%) |
| 선택됨 | `box-shadow: 0 0 0 2px #0c2948` + 옅은 글로우 |
| 연결된 선수과목 | `box-shadow: 0 0 0 2px #e8630a` + 좌상단 "선수과목" 주황 배지 |
| 비활성(학기 필터) | opacity 0.35, 클릭 불가 |

호버: 일반 카드 테두리 `#9db4d6` + 배경 `#f8fafd`, 필수 카드 배경 소폭 어둡게.
포커스: `outline: 2px solid #e8630a; offset 2px`.

## 4. 선수과목 관계 표현

- 기본 화면에서 연결선을 그리지 않는다 (원본의 복잡함 재현 금지).
- 상시 표기는 카드 내 "선수: ○○" 텍스트뿐.
- 카드 클릭 → 해당 카드 '선택됨', 선수과목 카드 '연결됨' 하이라이트. Esc로 해제.
- 스크린리더: 클릭 시 aria-live로 "공학수학 4의 선수과목: 공학수학 3 (2학년 1학기)" 낭독.
- (선택 구현) 1024px 이상에서만 선택 시 두 카드 사이 주황 2px 직교 연결선 1개.

## 5. 섹션별 내용

### MSC
- 1-1: 공학수학 1 / 공학물리학 및 실험 1 / 공학화학 및 실험 1 (모두 필수)
- 1-2: 공학수학 2 / 공학물리학 및 실험 2 / 공학화학 및 실험 2 (모두 필수)
- 2-1: 공학수학 3 / 공학정보처리 (모두 필수)
- 2-2: 공학수학 4 (필수, 선수: 공학수학 3 — 원본의 빨간 묶음 박스는 이 관계 표기로 대체)
- 3-1: 택1 그룹 { 공학수치해석, 확률 통계 }

### 전공
- 전공 기초 · 2학년 (8열 그리드 공유)
  - 2-1: 고체 역학(필수) / 열역학(필수) / 컴퓨터 응용 기계 설계
  - 2-2: 유체 역학 / 동역학 / 기계공학실험 (1) (모두 필수)
- 졸업 필수 스트립: 창의제품설계(필수) / 연구논문(필수)
- 세미나: 기계공학세미나 (1) / (2)
- 전공 심화 · 3~4학년: 좌우 2패널 ("1학기 개설" / "2학기 개설",
  제목은 네이비 + 파랑 밑줄 탭 — 주황 사용 금지)
  - 1학기 좌열: 재료거동학, 열 전달, 생산 공학, 응용 유체 역학, 기계 진동, 기계공학실험 (2)(필수), 학부연구 (1)
  - 1학기 우열: 환경기계공학, 마이크로 기계시스템, 전자기학및응용, 메카니즘설계, 바이오의료기계
  - 2학기 좌열: 응용고체역학, 응용열역학, 정형생산시스템, 에너지동력공학, 기계시스템제어, 메카트로닉스, 학부연구 (2)
  - 2학기 우열: 공학재료, 나노기계공학, 광공학, 기계요소설계, 생체역학, 컴퓨터 해석기반설계
- 스페셜 트랙: 과목 카드가 아닌 배너형 링크 (배경 `#eaf4fb`, "자세히 보기 →")

### 교양
- 좌 카드 "공통 교양 · 필수 이수": 글쓰기 / 기독교의 이해 영역 중 1과목 /
  대학영어 Ⅰ·Ⅱ (또는 고급대학영어 Ⅰ + 대학영어 Ⅱ) — 리스트로 분해
- 우 카드 "공학 관련 권장 교양": 과학기술과 사회, 기술창조와 특허, 테크노리더십,
  창업 103:21C, 기술 경영, 경제성 공학 — 칩 나열
- 학번 비교 (좌우 2카드):
  - 21학번 포함 이전: 7개 영역 칩 (정보와기술 포함) + "7개 영역 중 **5개 영역**에서 각 1과목씩 이수 필수"
  - 22학번 포함 이후: 6개 영역 칩 + "6개 영역 중 **4개 영역**에서 각 1과목씩 이수 필수"
  - 강조 스트립: 배경 `#dbeaf5`, 숫자만 20px/800 `#1769a8`
  - 하단 각주: "22학번부터 정보와기술 영역 제외, 이수 영역 5개→4개"

## 6. 반응형

- 1440px+: 8열 전부 노출 (그리드 최대 1320px), 가로 스크롤 없음
- 1024px: 열 간격 12px로 축소, 유지 불가 시 그리드 컨테이너만 가로 스크롤 (탭과 동기화)
- 768px 미만: 8열 폐기 → 학년 아코디언 (내부 학기 2열), 연결선 제거,
  진입 기준선은 아코디언 사이 가로 배지로 전환
- 390px: 완전 1열, 학기 탭 = 필터(딤이 아니라 해당 학기만 표시),
  3·4학년 1/2학기는 세그먼트 토글, 터치 영역 44px 이상

## 7. 접근성

- 시맨틱: section > 학기(h3) > 과목 목록(ul)
- 카드 aria-label: "공학수학 4, 2학년 2학기, 필수, 선수과목: 공학수학 3"
- 필수/택1은 색 + 텍스트 라벨 이중 표기 (색맹 대응)
- 키보드: 탭 순서 = 학기 탭 → 영역별 카드 학기순, Enter/Space 선택, Esc 해제

## 8. 구현 시 주의

- "창업 103:21C" 표기는 원본 그대로 유지하되, 학과 원자료로 정확한 과목명 확인 필요.
- 스페셜 트랙 구성 과목은 원본에 없음 — 별도 페이지 콘텐츠로 학과 확인 필요.
- 창의제품설계·연구논문의 개설 학기 표기 여부는 학과 확인 후 카드에 학기 칩 추가.
- 스크롤 트리거 애니메이션 금지. 하이라이트 전환 0.15s만 허용.
